예제

기본값을 5~49 로 바꾸면서 1.0 아래로 점수가 내려가기 시작하는 이웃의 개수 찾기

kn = KNeighborsClassifier()
kn.fit(fish_data,fish_target)

KNeighborsClassifier()
for n in range(5,50):
    kn.n_neighbors = n
    score = kn.score(fish_data,fish_target)
    if score < 1 :
        print(n, score)
        break

18 0.9795918367346939

n = 18 에서부터 1보다 작아진다
