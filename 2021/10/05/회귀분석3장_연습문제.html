[{r_{\text{xy}} = \frac{S_{\text{xy}}}{S_{x}S_{y}} = \frac{157.551}{3.7075 \times 18.07} = 2.3517 }{S_{\text{xy}} = \frac{1}{n - 1}\left{ \sum_{}^{}\text{xy} - \frac{\sum_{}^{}x\sum_{}^{}y}{n} \right} = 157.551 }{S_{x} = \sqrt{\frac{1}{n - 1}\left{ \sum_{}^{}x^{2} - \frac{\left( \sum_{}^{}x \right)^{2}}{n} \right}} = 3.7075 }{S_{y} = \sqrt{\frac{1}{n - 1}\left{ \sum_{}^{}y^{2} - \frac{\left( \sum_{}^{}y \right)^{2}}{n} \right}} = 18.07}]

정규방정식은 다음과 같다.

[b_{0}n + b1\sum_{}^{}x_{i} = \sum_{}^{}y_{i}]

[b_{0}\sum_{}^{}x_{i} + b1\sum_{}^{}{x_{i}}^{2} = \sum_{}^{}{x_{i}y_{i}}]

위에 주어진 값들을 대입하면

[{50b}{0} + 18b{1} = 3000 ]

b0 = 55.87 , b1=11.46 이다.

[{\widehat{y} = b_{0} + b_{1}\text{x\ }에\ 위에서\ 구한\ 값을\ 넣으면, }{\widehat{y} = 55.87 + 11.46 \times 3 = 90.25}]

[\sigma = MSE = \frac{\text{SSE}}{n - 2} = - 1510 ]
